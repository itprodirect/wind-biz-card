# Layout Spec (v0)

Card size: 3.5" x 2"
Bleed: 0.125"
Safe margin: 0.125"

Front hierarchy:
1) Name (largest)
2) Public Adjuster (FL & TX)
3) Phone + Email
4) LinkedIn text optional (QR usually on back)

Back hierarchy:
1) 1-line purpose / positioning
2) Support bullets (if included)
3) Xactimate Level 2 Certified
4) Licenses
5) BAS/SPC
6) QR bottom-right + label "LinkedIn"
